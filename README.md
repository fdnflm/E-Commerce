## Функционал

* Покупки
* Подписочная система для скидок
* Смена настроек
* Регистрация
* Авторизация
* Слияние корзины из сессии с БД
* Размещение промо-объявлений
* Отображение заказа(ов)
* Возможность скачать весь заказ
